# NextJS + Typescript + TailwindCSS

This is a barebones NextJS Typescript project with TailwindCSS.

### Install

```
npx tiged alexmnet/next-ts-tailwind <your-repo-name>
```

```
cd <your-repo-name>
```

```
npm install
```

```
npm run dev
```
